## How we spend sponsor funds

Sponsor funds pay for the maintainer's time. Specifically:

- **Maintenance** — dependency updates, security patches, CI fixes
- **Issue response** — triaging and reproducing bug reports, especially from atypical field environments
- **New features** — model adapters (MegaDetector, others), export formats (Wildlife Insights, iNaturalist), UI improvements based on researcher feedback
- **Community** — responding to WILDLABS posts, answering questions, helping researchers get the tool working with their specific dataset

There is no hosting cost (Studio is self-hosted by design), no paid infrastructure, and no profit margin. Every dollar goes to development time.
